Playwright Bingo Documentation

Overview

Playwright Bingo is a powerful testing framework built on top of Playwright and Cucumber, designed to make end-to-end testing more efficient and maintainable.

Key Features

  • Page Object Model for organized test structure
  • Cucumber Integration for BDD-style test writing
  • Data Masking for sensitive information protection
  • CLI Tools for project management

Getting Started

npm install playwright-bingo

After installation, you can initialize a new project using the CLI:

bingo init
